Nav <nav>

The HTML element repsents the section of a page whose purpose is toprovide navigation links, either within the current document or to other documents. For example, navigations sections are menus, tables of contents, and indexes.

CSS

Color

Element Example Description
hsl hsl(120, 10%, 12%)
  • Hue -is the first number. It refers to an angle on a color wheel. Red is 0 degrees, Green is 120 degrees, Blue is 240 degrees, and then back to Red at 360. Colour wheel
  • Saturation - refers to the intensity or purity of the color. The saturation increases towards 100% as the color becomes richer. The saturation decreases towards 0% as the color becomes grayer.
  • Lightness - refers to how light or dark the color is. Halfway, or 50%, is normal lightness. Imagine a sliding dimmer on a light switch that starts halfway. Sliding the dimmer up towards 100% makes the color lighter, closer to white. Sliding the dimmer down towards 0% makes the color darker, closer to black.
hsla, rbga, transparent - opacity
  • hsl(120, 10%, 12%, 0.1)
  • rgba(234, 45, 98, 0.33)
  • color: transparent
  • Fourth Param - between 0 and 1 represent the opacity
  • Frist 3 values work the same as hsl, the fourth is the alpha is sometimes called opacity
  • There is, however, a named color keyword for zero opacity, transparent. It’s equivalent to rgba(0, 0, 0, 0), and it’s used like any other color keyword
