Head

The HTML head is the contents of the <head> element. Not like the not like the <p> the contents of the head will not be displayed. Head contains meta data about the document. Things included in the head element.

Element Description Additional Attributes
<meta> Data that describes the data. Will tell who wrote it and the summary charset, content, http-equiv, name
<title> Title of the page that you would see in the tab header section

Heading Elements <h1> - <h6>

There are 6 levels of section headings <h1> to <h6>. There are no additional attributes.

Header <header>

<header> HTML element represents indrocucotry content, usually a group of navigational aids. It can also contain heading elements but also a log, a search form, an author name and other elements.

Nav <nav>

The HTML element repsents the section of a page whose purpose is toprovide navigation links, either within the current document or to other documents. For example, navigations sections are menus, tables of contents, and indexes.

Table <table>

Table is used to represent tablular data that is information presented in two dimensional table.
With the table attribute there are extra elements to create the table structure.

Element Description Additional Attributes
<th> Row Header
<tr> Row Description
<td> Cell Data Description
